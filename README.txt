Как использовать архиватор:

После того, как вы скомпилировали код архиватора и создали exe файл, необходимо в командной строке ввести следущее:

C:/USER/Debug/Achiver -pack -files C:/File_1 C:/File_2 C:/File_n -path C:/PATH

Где: USER - путь к exe файлу архиватора, File_n - файлы, которые нужно заархивировать, C:/PATH - путь, где будет создан архив с файлами под названием "binary_archiver".
На этом процесс архивации закончен, теперь про разархивацию:
Как и раньше вводим в командную строку следущее:

C:/USER/Debug/Achiver -unpack -files C:/PATH/binary_archiver -path C:/PATH_2

Где PATH_2 - путь, где появятся ранее заархивированные файлы.

!Важно! После слова "-files" путь к файлам необходимо указывать через "/", а не через "\", иначе будет ошибка создания файла.
