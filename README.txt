Как использовать архиватор:

После того, как вы скомпилировали код данного архиватора и создали exe файл, необходимо в командной строке указать следущее:

C:/USER/Debug/Archiver -pack -files C:/File_1 C:/File_2 C:/File_n -path C:/PATH

Где: USER - путь к exe файлу архиватора. File_n - файл, который нужно заархивировать. C:/PATH - путь, где в итоге будет создан архив с файлами под названием "binary_archiver".
На этом процесс архивации закончен, теперь про распаковку архива:
Как и раньше в командной строке вводим следущее:

C:/USER/Debug/Archiver -unpack -files C:/PATH/binary_archiver -path C:/PATH_2

Где: C:/PATH_2 - путь, где появятся файлы, которые прежде были заархивированы.

!Важно! Путь к файлам после слова "-files" необходимо указывать через "/", а не через "\", иначе будет ошибка создания файла.
